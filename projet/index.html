  • 2020-présent

    Site de téléchargement/cloud/convertisseur

    J'avais commencé tout seul le projet, puis deux amis m'ont rejoint, Raymond et Majeed.
    Le projet consiste à créer un site à la fois de cloud, téléchargement, de convertisseur de fichiers/url, ainsi que d'autres outils.
    Le site est herbergé sur ubuntu chez Raymond, il s'occupe de tout ce qui est materiel et du css.
    Majeed s'occupe du javascript.
    Je m'occupe du php, des commandes linux (pour la conversion de fichiers en différent format, par exemple), de la base de données, des requêtes Mysql et de la gestion du projet.
    Langages utilisés :
       - PHP
       - HTML
       - CSS
       - MySQL
       - Javascript
       - lignes de commande linux
